# OpenGraph Image Brief

## Canvas

- Size: 1280x640
- Format: PNG
- Purpose: GitHub social preview image for `gdocs-photo-report`

## Creative Direction

Create a polished, dark-themed dashboard mockup that communicates Google Docs automation, report assembly, and progress-aware batch processing at a glance.

## Layout

- Left third: a high-contrast product title block with the heading `Google Docs Photo Report`
- Subtitle below: `Automated image insertion for engineering reports`
- Small supporting line: `Google Apps Script + Drive folder picker + resumable batching`
- Right two-thirds: a stylized dashboard interface mockup showing report progress

## Visual Elements

- Dark charcoal background with soft blue gradients and subtle grid texture
- Floating interface cards with rounded corners and soft shadows
- A progress card that reads `Processing 45 of 150`
- A horizontal progress bar at roughly 30 percent complete
- A folder selection card showing a Drive folder name such as `Project 214 - Site Photos`
- A summary card with compact metrics: `Inserted`, `Skipped`, `Missing`, `Duplicates`
- Small Google Docs-inspired document preview with image thumbnails aligned above captions

## Style

- Premium SaaS aesthetic
- Clean typography with strong hierarchy
- Blue and cyan accent lighting, with restrained green for success states
- No clutter, no cartoon illustration, no mascots
- Emphasize precision, automation, and professional reporting

## Suggested Palette

- Background: `#0b1220`
- Surface: `#111a2b`
- Surface highlight: `#16233a`
- Primary accent: `#4f8cff`
- Secondary accent: `#35c6ff`
- Success accent: `#2ecc71`
- Text primary: `#f5f7fb`
- Text secondary: `#a8b3c7`

## Alt Text

Dark-themed dashboard mockup for Google Docs Photo Report showing Drive folder selection, a live progress bar, and image insertion metrics for an engineering report automation workflow.
